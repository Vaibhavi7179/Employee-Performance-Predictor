# 📊 Employee Performance Predictor (Google Colab)

## 🚀 Project Overview

This project builds a machine learning model to predict employee performance as **High, Medium, or Low** using HR-related features like experience, training hours, attendance, and productivity.

---

## 🎯 Objective

* Predict employee performance
* Help HR teams make data-driven decisions
* Identify high and low performers

---

## 🧠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn
* Google Colab

---

## 📂 Project Structure

```
Employee-Performance-Predictor/
│
├── Employee_Performance_Predictor.ipynb   # Main Colab Notebook
├── README.md
```

---

## ▶️ Run in Google Colab

1. Open Google Colab
2. Upload the notebook OR open from GitHub
3. Run all cells step-by-step

---

## ⚙️ Steps in Notebook

1. Install libraries
2. Create synthetic dataset
3. Perform EDA (visualization)
4. Preprocess data
5. Train model (Random Forest)
6. Evaluate performance
7. Make predictions

---

## 📊 Output

* Performance classification (High / Medium / Low)
* Accuracy score
* Confusion matrix
* Visualization graphs

---

## 🧪 Sample Features

## 📈 Results
